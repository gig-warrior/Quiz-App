## Quiz App
A responsive Quiz Application built using HTML, CSS and JavaScript. The app fetches random quiz questions from an online API and lets users test their knowledge with a timer and score tracking.

## Live Demo
[Try the Quiz App](https://gig-warrior.github.io/Quiz-App/)
